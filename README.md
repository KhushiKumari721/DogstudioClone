# Dogstudio Clone
The clone focuses on building a visually engaging landing page that demonstrates how modern creative agency websites are designed and implemented. It highlights smooth scrolling effects, interactive UI components, and a responsive layout that adapts seamlessly across different screen sizes including desktops, tablets, and mobile devices.
The primary objective of this project is to recreate the visual style, layout structure, and interactive experience inspired by the original Dogstudio website.

The project emphasizes clean code structure and component-based development. By organizing the interface into reusable components, the application becomes easier to maintain, scale, and improve. This approach reflects the development practices used in modern frontend applications.
Another important aspect of this project is the implementation of modern design principles such as minimalistic layouts, dynamic transitions, and visually appealing content sections. These elements help create a more immersive browsing experience for users.

This project also serves as a practical learning exercise for improving skills in frontend development, responsive web design, and UI structuring. Recreating a real-world website design helps in understanding how professional websites are structured and how interactive user experiences are built using modern technologies.
Overall, this project demonstrates the ability to build modern, responsive, and visually appealing web interfaces using current frontend tools and development practices.

## Features
- Modern and visually engaging landing page  
- Smooth scrolling and interactive UI elements  
- Responsive design for mobile, tablet, and desktop  
- Clean and structured component-based layout  
- Interactive sections and smooth transitions  

## 🛠️ Technologies Used
- React.js
- Tailwind css
- Three js
- GSAP
